# Hospital Management SQL Assignment

## Overview

This repository contains my **Hospital Management SQL Assignment**, completed in **July 2026**. The project demonstrates the design and implementation of a relational database for managing hospital operations using SQL.

The assignment showcases core database concepts such as database design, table creation, relationships, data manipulation, querying, views, triggers, indexing, and referential integrity.

## Features Implemented

This project includes:

* Database creation
* Table creation with appropriate data types
* Primary and Foreign Keys
* Data insertion and updates
* SQL queries using:

  * `SELECT`
  * `WHERE`
  * `ORDER BY`
  * `GROUP BY`
  * `HAVING`
* Aggregate functions
* SQL Joins
* Nested queries (Subqueries)
* Views
* Triggers
* Indexes
* Cascading deletes using Foreign Key constraints

## Database Entities

The database models a hospital management system and includes entities such as:

* Patients
* Doctors
* Appointments
* Treatments
* Bills

These entities are linked through relational keys to ensure data consistency and integrity.
